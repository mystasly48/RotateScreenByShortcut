# Rotate Screen by Shortcut


|          Hotkey          |       Behavior        |
| ------------------------ | --------------------- |
| Ctrl + Alt + Up Arrow    | Rotate to 0 degrees   |
| Ctrl + Alt + Right Arrow | Rotate to 90 degrees  |
| Ctrl + Alt + Down Arrow  | Rotate to 180 degrees |
| Ctrl + Alt + Left Arrow  | Rotate to 270 degrees |
